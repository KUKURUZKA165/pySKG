# pySKG
### <b>Rofl</b> steam keys generator script with some cool features.

### [Download here](https://codeload.github.com/KUKURUZKA165/pySKG/zip/refs/heads/main)

### Installation:

Install dependecies:
```
pip install steam
```

Launch:
```
python pySKG.py
```
